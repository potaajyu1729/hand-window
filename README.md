# hand-window
カメラ越しに“空間を切り取る”体験を実現するWebアプリ

---

## 概要

このプロジェクトは、Webカメラと手のジェスチャー認識を使い、
現実の映像の中に「仮想の窓（ポータル）」を作り出すインタラクティブWebアプリです。

両手でフレームを作ることで、その領域だけ別の映像（異世界映像）が表示されます。

---

## 体験できること

- カメラ映像のリアルタイム表示
- MediaPipeによる手の骨格認識
- 両手の指を使った四角形ジェスチャー
- その領域だけに別動画を表示（ポータル表現）

---

## 使用技術

- HTML / CSS / JavaScript
- MediaPipe Hands（手の骨格推定）
- Canvas API
- WebRTC（カメラ映像取得）

---

## 注意

- カメラアクセスが必要です
- HTTPS環境（GitHub Pagesなど）での実行を推奨
- PCブラウザ（Chrome推奨）での利用を想定
